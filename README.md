## 功能说明

好啊有保留 Legado / 阅读 Max 的主要阅读能力，包括：

- 自定义书源
- 书架管理
- 搜索与发现
- 本地 TXT / EPUB 阅读
- 阅读界面自定义
- 替换净化
- 订阅内容
- 多种翻页模式
- WebDAV / 本地备份恢复
- 主题与阅读配置管理
